Executive Summary
Budget Tracker is a lightweight, fast, and fully responsive financial management application
designed to help users track income, expenses, and categories in real-time. Built with a modern
React + Node architecture, it features a clean UI, instant calculations, and dynamic analytics.
Core Features
Dashboard: Live income, expenses, balance, category breakdown chart, and monthly comparison.
Transactions: Add/view/delete transactions with validation and PKR formatting.
Insights: Real-time category analytics and visual charts using Recharts.
Tech Stack
Frontend: React, TailwindCSS, React Router, Recharts, Axios.
Backend: Node.js, Express, CORS, in-memory storage.
How to Run
Backend:
cd backend
npm install
npm run dev
Frontend:
cd frontend
npm install
npm start
Highlights
Fast SPA, clean UI/UX, fully responsive, in-memory API, real-time charts, PKR currency support.
Future Enhancements
Database integration, user authentication, data export, budget planning features.
